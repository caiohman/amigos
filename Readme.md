### Amigos


All documentation necessary are stored in doc folder.

First of all it is necessary to start mysql container and populate it. 

In order to run dotnet backend:

1. Go to folder bk

2. run command `dotnet run`


In order to run on mobile devices:

1. Go to folder mobile

2. run command `flutter run -d <device name>`


In order to run desktop app:

1. Go to folder app

2. run command `cargo run`
